Source: St. Petersburg Programming Team Championship 2017


# Problem

Given N ≤ 40, arrange from 1 to N^2 like this:

```
1 2 3 ... N
N+1 N+2 N+3 ... 2N
...
... N^2
```

Print each number as `*` if it has ≤ k divisors, otherwise `.`.
